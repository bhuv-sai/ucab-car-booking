# Ucab - Car Booking Application 🚖

Ucab is a mobile car booking application developed using React Native. The application provides users with a simple and convenient way to book rides, view nearby available vehicles, estimate fares, and manage their bookings from a mobile device.

## Features

* User authentication and registration
* Cab booking with pickup and drop locations
* Multiple vehicle categories
* Estimated fare calculation
* Nearby cab availability
* Ride history management
* User profile management
* Responsive mobile interface
* Real-time ride updates

## Technology Stack

### Frontend

* React Native
* React Navigation
* Native Base
* React Native Vector Icons

### Backend Services

* REST APIs
* JSON-based data handling

## Installation

Clone the repository:

```bash
git clone https://github.com/bhuv-sai/ucab-car-booking.git
cd ucab-car-booking
```

Install dependencies:

```bash
npm install
```

Start the Metro bundler:

```bash
npx react-native start
```

Run on Android:

```bash
npx react-native run-android
```

Run on iOS:

```bash
npx react-native run-ios
```
